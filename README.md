# Doctor Appointment Booking App (React)

This project is a simple doctor appointment booking application built using React.  
I created this project to practice handling forms, validations, and real-world UI logic using React hooks.

The application allows users to book, update, and delete doctor appointments with confirmation messages.

---

## Features

- Book a new appointment
- Update existing appointment details
- Delete appointments with confirmation
- Input validation for:
  - Required fields
  - Phone number format
  - Time format (AM / PM)
- Date and time formatting
- Toast messages for confirmation and information
- Simple action menu (Edit / Delete)

---

## Tech Stack

- React
- JavaScript
- CSS
- Font Awesome Icons

---

## How the Application Works

1. User fills in the appointment form
2. All fields are validated before submission
3. Appointments are displayed in a table
4. Each appointment can be edited or deleted
5. Confirmation popup appears before update or delete
6. Data is managed using React state
